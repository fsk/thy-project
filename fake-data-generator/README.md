# fake-data-generator

A simple Python CLI application that generates fake user data.

## Setup

```bash
python3 -m venv .venv
source .venv/bin/activate
pip install -r requirements.txt
```

## Running

```bash
python main.py --count 5 --format pretty
```

For JSON output:

```bash
python main.py --count 10 --format json
```
